# Website Scraper using BeautifulSoup

A robust, automated web scraping application built in Python utilizing the `BeautifulSoup4` and `requests` libraries. This tool safely establishes connection streams with target servers, navigates DOM structures, extracts specific data parameters (such as headlines and prices), and stores them into structured, tabular CSV datasets for data analysis.

## Key Features
* **Automated Data Retrieval:** Uses network requests to safely fetch remote HTML web pages.
* **Intelligent DOM Parsing:** Leverages BeautifulSoup tree-navigation to locate nested container structures.
* **Data Cleaning & Filtering:** Automatically sanitizes raw text fragments, removing white spaces and formatting characters.
* **Structured Local Storage:** Exports compiled information directly into clean, standardized CSV files.
* **Error Handling:** Built-in network exception blocks and element validation rules to handle unexpected structural page updates seamlessly.

## Project Structure
```text
├── scraper.py          # The core Python application script containing parsing logic
├── README.md           # Project documentation and architectural overview
└── scraped_data.csv    # Final structured output dataset generated by the application
```

## System Process Flow
1. **Network Initialization:** Configures client request payloads and target URLs.
2. **HTTP GET Handshake:** Dispatches request streams and validates status responses (200 OK code).
3. **HTML Structural Tree Compilation:** Converts flat text strings into traversable node formats.
4. **Target Element Filtering:** Finds designated layout nodes via specific HTML tags and class identifiers.
5. **Schema Layout Writing:** Maps unstructured parsed arrays directly into a uniform data sheet rows layout.

## Setup and Installation Instructions

### Prerequisites
Ensure you have Python 3.x installed on your computer.

### 1. Install Required Libraries
Open your terminal or command prompt and run the following command to download dependencies:
```bash
pip install requests beautifulsoup4
```

### 2. Execute the Scraper
Run the application using the following execution command:
```bash
python scraper.py
```

### 3. Review Generated Dataset
Once processing concludes successfully, look inside your current directory path to find `scraped_data.csv`. This sheet can be viewed in Microsoft Excel, Google Sheets, or any programmatic pandas environment.
